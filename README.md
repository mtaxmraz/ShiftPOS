# ShiftPOS
A personal project of mine that I want to work on. This will be both focused on the backend and the frontend

**Backend:**
 Supabase with Postgres, Auth, Row Level Security, and Realtime

 **Frontend:**
 Expo, React Native, and TypeScript

 **Payments Later:**
 Stripe / Stripe Terminal

 **Deployment:**
 Expo Application Services

Features that I want Implemented into the code:
1. Authentication
   - sign up
   - Log in
   - Log Out
   - Basic User Information
2. Restaurant Setup
   - Create a restaurant
   - Add users to the restaurant
   - Assigning roles to the users
   - Create and Edit a Table System
3. Menu
   - Creating Categories
   - Adding menu items
   - Editing Prices
   - Marking items as 86'd
   - Inventory count
4. Order Creations
   - Selecting a menu item
   - Creating a order
   - Order being snet to the Kitchen
5. Kitchen Display System (KDS)
   - Kitchen Staff sees active orders
   - Orders update in real time
   - Kitchen can mark orders as
     - New
     - In Progress
     - Ready
     - Completed
6. Reports
   - Daily Order count
   - Daily Revenue estimate
   - Most ordred item(s)
